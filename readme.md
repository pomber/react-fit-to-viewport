Automatically scales the content to fit it inside the viewport. It also works with server-side rendering.

```bash
$ yarn add react-fit-to-viewport
```

```jsx
<FitToViewport width={800} height={600} minZoom={0} maxZoom={1}>
  {stuff}
</FitToViewport>
```
